Политика в отношении обработки персональных данных

1. Общие положения

Настоящая политика обработки персональных данных составлена в соответствии с требованиями Федерального закона от 27.07.2006 № 152-ФЗ «О персональных данных» и определяет порядок обработки персональных данных и меры по обеспечению безопасности персональных данных, предпринимаемые ООО "АГИДЕЛЬ".

1.1. Оператор ставит своей важнейшей целью соблюдение прав и свобод человека при обработке персональных данных, включая защиту частной жизни.

1.2. Политика применяется ко всей информации, которую Оператор может получить о посетителях сайта https://vipavto.online.

2. Основные понятия

  • Автоматизированная обработка: обработка с использованием вычислительной техники.
  • Блокирование: временное прекращение обработки данных.
  • Веб-сайт: https://vipavto.online.
  • Информационная система: совокупность баз данных и технологий обработки.
  • Обезличивание: невозможность определить принадлежность данных субъекту.
  • Обработка: любые операции с персональными данными.
  • Оператор: лицо, осуществляющее обработку данных.
  • Персональные данные: информация о пользователе.
  • Пользователь: посетитель сайта.
  • Передача: раскрытие данных третьим лицам.
  • Распространение: раскрытие неограниченному кругу лиц.
  • Трансграничная передача: передача за границу.
  • Уничтожение: безвозвратное удаление данных.

3. Права и обязанности Оператора

3.1. Оператор имеет право:

  • получать достоверные данные от субъекта;
  • продолжать обработку при наличии законных оснований;
  • самостоятельно определять меры защиты.

3.2. Оператор обязан:

  • предоставлять информацию субъекту;
  • обрабатывать данные согласно закону;
  • отвечать на запросы;
  • обеспечивать защиту данных;
  • прекращать обработку при необходимости.

4. Права и обязанности субъектов данных

4.1. Субъекты имеют право:

  • получать информацию об обработке;
  • требовать исправления или удаления данных;
  • отзывать согласие;
  • обжаловать действия Оператора.

4.2. Обязанности:

  • предоставлять достоверные данные;
  • обновлять информацию.

5. Принципы обработки данных

  • законность и справедливость;
  • ограничение целей;
  • минимизация данных;
  • точность и актуальность;
  • ограничение сроков хранения.

6. Цели обработки

  • Цель: информирование пользователя по email
  • Данные: ФИО, телефон
  • Основание: законодательство РФ
  • Действия: передача данных

7. Условия обработки

  • с согласия пользователя;
  • для выполнения закона;
  • для исполнения договора;
  • для защиты интересов;
  • для общедоступных данных.

8. Порядок обработки и хранения

Оператор обеспечивает безопасность данных и предотвращает несанкционированный доступ.

  • данные не передаются третьим лицам без оснований;
  • пользователь может обновить данные по email;
  • срок хранения определяется целями обработки;
  • возможен отзыв согласия.

9. Действия с данными

  • сбор, хранение, использование;
  • передача и удаление;
  • автоматизированная обработка.

10. Трансграничная передача

Оператор уведомляет органы и получает необходимые сведения перед передачей данных за границу.

11. Конфиденциальность

Данные не раскрываются третьим лицам без согласия, если иное не предусмотрено законом.
